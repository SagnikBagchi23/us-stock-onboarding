# US Stocks Onboarding

A high-fidelity Expo prototype for the Groww Invest US stocks onboarding flow, built with the Mint design system.

## Stack

- **Expo** (managed workflow) + **Expo Router** (file-based navigation)
- **React Native** 0.81 / **React** 19
- **Reanimated 4** for animations
- **TypeScript**

## Screens

- **Stock detail** — chart, performance, holdings card, about section
- **Personal details** — employment, income, net worth, and source of funds selectors with animated bottom sheets

## Design System

Groww Mint DS dark tokens — semantic color and typography tokens, Sohne (headings) and Groww Sans (body), Hugeicons.

## Getting Started

```bash
npm install
npm start        # Expo dev server
npm run web      # Web (browser)
npm run ios      # iOS simulator
npm run android  # Android emulator
```

## Build

```bash
npm run build    # Export static web build
```
